# Book Identity v3.13 — Mitr + Mobile Fix

รอบนี้แก้ 2 จุดตามที่ขอ

## 1. ฟอนต์

หัวข้อและข้อความตัวหนาทั้งหมดเปลี่ยนเป็น `Mitr`

รวมถึง:

- หน้าแรก
- Prelude
- Chapter
- คำถาม
- Loading
- Unlock
- Result
- ชื่อ Identity
- ปุ่ม
- Eyebrow / Label
- ข้อความตัวหนา

## 2. บัคมือถือ

สาเหตุคือวลีที่บังคับให้อยู่บรรทัดเดียวใช้ `white-space: nowrap`
ทำให้บางหน้าจอมือถือข้อความล้นหรือถูกตัด

แก้แล้วโดย:

- Desktop ยังจับวลีให้อยู่บรรทัดเดียว
- Mobile ปล่อยตัดบรรทัดอัตโนมัติ
- ป้องกันข้อความล้นกล่อง
- ปรับขนาดหัวข้อบนมือถือให้พอดี
