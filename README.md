# Book Identity v3.1 — Unlock Fixed

เวอร์ชันนี้ทำให้หน้าปลดล็อกแตกต่างจากหน้าผลลัพธ์อย่างชัดเจน และใส่ cache-busting เพื่อป้องกัน GitHub Pages แสดงไฟล์เก่า

## วิธีเช็กว่าไฟล์ใหม่ทำงาน

หน้าแรกด้านล่างต้องเห็นคำว่า:

`build 3.1`

หลังตอบครบ 8 ข้อ Flow ต้องเป็น:

1. Loading
2. หน้าพื้นหลังสีเข้ม
3. หนังสือปิดพร้อมกุญแจ 3 จุด
4. ปุ่ม “คลิกเพื่อปลดล็อกผลลัพธ์”
5. กดแล้วกุญแจหลุดและค่อยแสดง Result

## วิธีอัป GitHub ให้ถูก

1. ลบไฟล์เดิมใน Repository หรือเลือก Replace ทุกไฟล์
2. อัปไฟล์ด้านในแพ็กนี้ทั้งหมด
3. ให้ `index.html` อยู่หน้าแรกของ Repository
4. รอ Pages Deploy สำเร็จ
5. เปิดเว็บแล้วกด `Ctrl + F5`
6. บนมือถือให้ปิดแท็บเดิม แล้วเปิดลิงก์ใหม่

หากหน้าแรกไม่เห็น `build 3.1` แปลว่ายังเป็นไฟล์เก่าหรือแคชเก่า
